GM Chat7 Management

Use these tools to manage all Cyberpunk Agent data across all devices in the registry.

Clear All Messages

This will delete ALL chat message histories for ALL devices in the registry. This action cannot be undone.

Clear All Contact Lists

This will clear the contact list from all agent devices in the registry. This will remove all contacts but keep the devices themselves. This action cannot be undone.

Synchronize All Devices

This will synchronize all devices with the registry to ensure all devices have phone numbers attached to them. This will update any devices that are missing phone numbers.

Master Reset System

This will completely reset the entire Cyberpunk Agent system: clear ALL messages, ALL contact lists, and force ALL clients to restart. This action is IRREVERSIBLE and will affect ALL users!

Warning: These actions are irreversible and will affect all users in the game.
